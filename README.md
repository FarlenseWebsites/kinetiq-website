KinetiQ — web front-end for KinetiQ Physiotherapy.

## Quick Start

Install dependencies and run the dev server:

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000 to view the site.

## Documentation
- Developer guide: see [docs/DEVELOPER_GUIDE.md](docs/DEVELOPER_GUIDE.md)
